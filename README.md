# Digital-Bank-Landing-Page
Digital Bank Landing Page is a responsive single-page website designed for businesses to showcase their work interactively. Built with HTML, CSS, and JavaScript, it includes starter files for easy setup, allowing users to customize and share their own versions.

## Features
- Fully responsive design
- Interactive elements created by JavaScript
- A layout suitable for business portfolios
- Starter file provided for easy project setup and customization
